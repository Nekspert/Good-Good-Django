---

## Зачем нужен фреймворк Django?

Даже при разработке **простых сайтов** приходится каждый раз разрабатывать однотипные программные модули:

- для обработки и отправки клиенту HTML-страниц;
- взаимодействия с базой данных;
- создания админ-панели;
- регистрации и авторизации пользователей;
- и многое другое...

Чтобы **не повторять одни и те же действия** и **автоматизировать стандартные алгоритмы** работы серверной части сайтов, были придуманы **фреймворки** — готовые решения для разных языков программирования.

👑 Для языка **Python** лидером является **Django**.

> Разработчики таких сервисов, как **YouTube**, **Google Search**, **Dropbox** уже давно используют Django как **надежное**, **простое** и **быстрое** средство обеспечения бесперебойной работы своих сайтов.

И самое главное — Django является **открытым и бесплатным инструментом**, доступным каждому.

Теперь каждый может **строить профессиональные сайты любой сложности**.  
Осталась самая малость — **изучить Django!**
